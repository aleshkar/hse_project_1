Класс для управления базой данных финансового Telegram-бота Обеспечивает все операции с SQLite базой данных: создание таблиц, управление балансом пользователей, работу с расходами и статистикой.

Основные методы: - create_tables(): Создает структуру базы данных - set_balance(): Устанавливает/обновляет баланс пользователя - get_balance(): Получает текущий баланс пользователя - add_expense(): Добавляет расход с проверкой средств - get_stats(): Возвращает статистику по категориям - get_history(): Возвращает историю расходов - clear_data(): Полностью удаляет данные пользователя
